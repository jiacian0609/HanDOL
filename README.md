## Backend
### Install Packages
```shell=
npm install
cd backend
npm install
```
### Start Backend
```shell=
cd backend
npm start
```

## Frontend
### Install Packages
```shell=
cd frontend
npm install
```
### Start Frontend
If you are going to use the backend server that runs on the localhost, open `/frontend/src/api.js` and change the `hostname` into `http://localhost:3000`.
```shell=
cd frontend
npm start
```
